# 🎓 Student Grade Tracker

A simple Java console application that allows users to store, manage, and analyze student grades. This project demonstrates the use of ArrayList, loops, conditionals, and methods in Java.

---

## 📌 Description

The Student Grade Tracker is a menu-driven program that enables users to:

- Add student grades
- Compute the average grade
- Find the highest grade
- Find the lowest grade
- Display all entered grades

This project is ideal for beginners learning Java fundamentals such as object-oriented programming, collections, user input handling, and method creation.

---

## 🚀 Features

✔ Add multiple student grades  
✔ Calculate the average grade  
✔ Identify the highest grade  
✔ Identify the lowest grade  
✔ Display all stored grades  
✔ User-friendly menu interface  
✔ Handles empty list scenarios  

---

## 🛠 Technologies Used

- Java
- ArrayList
- Scanner
- Control flow statements (loops & conditionals)
